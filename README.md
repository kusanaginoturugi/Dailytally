# Dailytally

「毎日集計」用のシンプルなWebアプリです。

## できること

- 9つの伝道会ごとに、1週間分（7日）の数字を入力
- 7〜9項目を切り替えて入力（既定は9項目）
- 合計ページで、9伝道会分の日次合計と週間合計を確認
- 週の開始日を変更して運用可能
- 得道者数開始日を指定して、合計ページの見出しに反映
- 目標数は週の初日のみ入力可能
- 入力内容はブラウザの `localStorage` に自動保存
- 以前の保存形式（`daily-tally-v1`）がある場合は初回起動時に自動移行
- 最後に開いていたタブ（各伝道会/合計ページ）も保存して復元

## 使い方

1. `index.html` をブラウザで開く
2. 画面上部で「週の開始日」「項目数（7/8/9）」「得道者数開始日」を設定
3. 上部タブから各伝道会名を開いて入力
4. 「合計ページ」タブで全伝道会合計を確認
